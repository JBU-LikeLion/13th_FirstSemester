# 이미지 분류기 과제

## 과제 공지일
**2025년 5월 27일자 과제**입니다.

## 과제 내용
기존 이미지 분류기 정확도: 약 66~71%

다양한 방법을 시도하여 정확도 높여보기 
정확도 목표 **72% 이상**!! 

### 필수 항목
1. 정확도를 높이기 위해 시도한 방법과 과정을 노션에 서술 (사진 필수)
2. 정확도 그래프 (matplotlib 시각화)
3. 테스트 결과 (최소 2개 이상)

### 힌트
1. 모델 변경(MoblieNet, VGG 등)
2. 애폭(epoch) 변경
3. EarlyStopping, ModelCheckpoint 등을 이용해 학습 효율 향상 가능
